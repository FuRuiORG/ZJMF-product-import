# 智简魔方上游产品批量导入工具

Author: FuRuiORG/RuiNexus

>当前只建议使用一级分组导入

## 功能介绍

本工具用于批量导入智简魔方(ZJMF)上游产品，支持以下功能：

- 登录后台获取会话
- 获取上游供货商列表
- 选择上游后获取其产品分组(一级/二级)
- 用户选择分组后批量导入
- 自动创建分组(带上游拼音首字母前缀，如 `MD-香港`)
- 导入后自动根据折扣创建客户分组

## 环境要求

- Python 3.6+
- 依赖库：`requests`

可选依赖（用于自动拼音转换）：
```bash
pip install pypinyin
```

## 安装

```bash
# 克隆或下载项目
cd product_import

# 安装依赖
pip install requests

# 可选：安装拼音转换库
pip install pypinyin
```

## 使用方法

### 运行脚本

```bash
python product_import.py
```

### 操作流程

```
1. 输入网站后台地址
   - 示例：https://xxx.com

2. 输入后台路径
   - 示例：admin2024

3. 输入用户名和密码
   - 登录成功后会提示是否保存配置

4. 选择上游供货商
   - 显示所有可用上游列表
   - 输入编号选择

5. 选择导入模式
   - [1] 按一级分组导入：选择一级分组，导入其下所有二级分组和产品
   - [2] 按二级分组导入：直接选择二级分组导入

6. 选择要导入的分组
   - 支持多选（用逗号分隔）
   - 输入 0 导入全部分组

7. 确认导入
   - 显示将要导入的分组信息
   - 确认后开始导入

8. 自动处理折扣分组
   - 导入完成后自动获取每个产品的折扣
   - 创建对应折扣分组（如 60、70 等）
   - 将产品添加到对应折扣分组
```

## 配置文件

首次运行后，可选择保存配置到 `config.json`：

```json
{
  "base_url": "https://xxx.com",
  "admin_path": "admin2024",
  "username": "admin"
}
```

配置选项：
- 保存(不含密码)：下次需要手动输入密码
- 保存(含密码)：密码明文存储，注意安全风险
- 不保存：每次都需要输入完整信息

## API 接口说明

本工具使用以下接口：

| 功能 | 接口 | 方法 |
|------|------|------|
| 登录 | `/后台路径/login` | POST |
| 获取上游列表 | `/后台路径/zjmf_finance_api` | GET |
| 获取上游产品 | `上游URL/v1/products` | GET |
| 获取本地产品 | `网站URL/v1/products` | GET |
| 创建一级分组 | `/后台路径/save_product_first_group` | POST |
| 创建二级分组 | `/后台路径/save_product_group` | POST |
| 导入产品 | `/后台路径/zjmf_finance_api/inputproduct` | POST |
| 获取产品折扣 | `/后台路径/product/get_upstream_price` | GET |
| 获取折扣分组 | `/后台路径/product/productgroup` | GET |
| 创建折扣分组 | `/后台路径/product/add_productgroup` | POST |
| 添加产品到分组 | `/后台路径/product/edit_productgroup` | POST |

## 分组命名规则

### 一级分组
格式：`上游拼音首字母-分组名`

示例：
- 上游名称：美得云
- 原分组名：香港
- 新分组名：`MD-香港`

### 折扣分组
格式：折扣百分比数字

示例：
- 60% 折扣 → 分组名：`60`
- 70% 折扣 → 分组名：`70`

## 注意事项

1. **SSL证书**：工具默认忽略SSL证书验证，适用于自签名证书环境

2. **拼音转换**：
   - 已安装 `pypinyin`：自动转换中文到拼音首字母
   - 未安装：手动输入缩写

3. **导入价格**：默认使用原价(100%)导入

4. **批量导入**：每批最多导入50个产品

5. **折扣处理**：
   - 导入完成后逐个查询产品折扣
   - 折扣 < 100% 的产品会加入对应分组
   - 同一产品只能属于一个折扣分组

## 常见问题

### Q: 登录失败怎么办？
A: 检查以下几点：
- 网站地址是否正确（不要包含后台路径）
- 后台路径是否正确
- 用户名密码是否正确
- 网络是否通畅

### Q: 创建分组成功但无法继续？
A: 工具会自动查询分组列表获取ID，如果仍然失败，请检查API返回格式是否与预期一致。

### Q: 折扣分组创建失败？
A: 可能是分组名称已存在但工具未检测到，尝试重新运行或手动检查分组列表。

### Q: 如何查看导入结果？
A: 登录后台查看产品管理，检查分组和产品是否正确创建。
